UC 4.05 Improving the Grid Infrastructure

Model / SGAM Business Layer / Business use cases / UC 4.05 Improving the Grid Infrastructure

Project:
  • DSO provides expected end of life for distribution grid assets to Grid Planning software.
  • DSO provides historical electrical data,new RES connection requests, new charging station locations to Grid Planning software
  • Grid Planning software access historical data on energy prices and stored data on equipment installation costs.
  • Grid planning software computes long term load forecast and RES generation.
  • Grid Planning software computes the optimal grid upgrades in terms of costs and RES hosting capacity.
  • Grid Planning software provides the necessary for data visualization -data export to DSO.
